# Client & Contact CRM

> People and relationship management

## Quick Commands
- "Add contact [name]" → Create new contact file
- "Prep me for [name]" → Get full context before a call
- "Log call with [name]" → Record meeting notes
- "Who is [name]?" → Pull up their profile

## Contact Template

When adding a new contact, I create a file in `clients/` with:

```markdown
# [Name]

## Basic Info
- **Company:** 
- **Role:** 
- **Email:** 
- **Phone:** 
- **LinkedIn:** 

## Relationship
- **Type:** Client / Prospect / Partner / Vendor / Other
- **How we met:** 
- **Introduced by:** 

## Preferences
- **Communication:** Email / Phone / Text
- **Best time to reach:** 
- **Notes:** 

## History
| Date | Type | Notes |
|------|------|-------|
| | | |

## Context for Calls
*Key things to remember before talking to them*

## Open Items
- [ ] 
```
